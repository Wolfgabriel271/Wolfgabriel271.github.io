## 👤 About

This is the personal portfolio website of **Jazen Gabriel M. Digamon**, a BS Computer Engineering student at the **University of Bohol**, Philippines. Built from scratch using pure HTML, CSS, and JavaScript — no frameworks, no templates — just clean, handwritten code.

The site showcases my skills, projects, certifications, and background as I work toward my goal of becoming a **Senior Software Engineer**.

## 🛠️ Built With

- **HTML5** — Semantic structure across 6 pages
- **CSS3** — Custom properties, glassmorphism, animations, light/dark theme, fully responsive
- **JavaScript (Vanilla)** — Typewriter, particle canvas, smooth scroll (Lenis), scroll reveal, project modal, theme toggle, hamburger menu
- **Font Awesome 6** — Icons
- **Google Fonts** — Playfair Display, DM Sans, DM Mono

## 🚀 Projects Featured

### Foursight Car Rental Management System
A full-featured C console application supporting up to 400 cars, 1,000 customers, and 2,000 rental records. Features role-based login, live dashboard, automated late fee calculation, barcode generation, email notifications via libcurl, and ANSI color UI.

### Kape Inato Cafe Management System
A full-stack PHP/MySQLi web system for a real local café in Bohol. Features an online ordering portal, admin dashboard, inventory management, QR ordering, sales analytics, and SMTP email confirmations — hosted on InfinityFree.
