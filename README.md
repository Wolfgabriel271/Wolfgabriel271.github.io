# Jazen Gabriel M. Digamon — Online Portfolio
### `wolfgabriel271.github.io`

> *"Hardware gives you the laws of physics. Software gives you the freedom to break the rules within them. Every great system starts at the silicon."*

A personal portfolio website themed around **Elden Ring**, built entirely with vanilla HTML, CSS, and JavaScript — no frameworks, no build tools, just handwritten code with a dark-fantasy soul.

---

## 👤 About

This is the personal portfolio website of **Jazen Gabriel M. Digamon**, a BS Computer Engineering student at the **University of Bohol**, Philippines. The site showcases my background, technical skills, capstone and personal projects, certifications, and contact information — wrapped in an atmospheric Elden Ring–inspired interface, as I work toward my goal of becoming a **Senior Software Engineer** and **Embedded Systems Engineer**, bridging both software and hardware.

## 🛠️ Built With

- **HTML5** — Semantic structure across 6 pages, with SEO meta tags and Open Graph data on every page

## ✨ Features

- 🌗 **Light / Dark mode** toggle with `localStorage` persistence
- 📱 **Fully responsive** — optimized for desktop, tablet, and mobile
- ☰ **Hamburger navigation** with slide-in drawer for mobile devices
- 🌫️ **Living background** — animated gold moon parallax, drifting ashen fog, floating rune symbols, rising embers
- ✍️ **Typewriter effect** cycling through roles (Computer Engineering Student, Embedded Systems Engineer, Hardware & Firmware Developer, Microcontroller & IoT Enthusiast, etc.)
- 📊 **Animated counters** and Erdtree-gold skill progress bars
- 🖼️ **Project modal** with tech stack badges and feature breakdowns
- 🏅 **Certificate gallery** with active-state thumbnail switching
- 🎞️ **Scroll reveal animations** site-wide via Intersection Observer
- ⬆️ **Scroll-to-top button** and smooth scrolling (Lenis)
- ♿ **Accessibility** — skip-to-content link, `aria-label`s, focus states, `prefers-reduced-motion` support
- ⚡ **Performance-aware** — animation loops pause on hidden tabs (Page Visibility API)
- 🎮 **Easter egg** — a hidden "Lost Grace" surprise for curious visitors

## 🚀 Projects Featured

### Foursight Car Rental Management System
A comprehensive C console application supporting up to 400 cars, 1,000 customers, and 2,000 rental records via flat-file storage. Features role-based login, a live dashboard, automated late fee calculation, barcode generation (via `zint` CLI), SMTP email notifications (via `libcurl`), and an ANSI-colored terminal UI.

**[📁 Source Code](https://drive.google.com/drive/u/1/folders/1zpuuSNAwoMHz94zhKZpZjiJakFftrMV_)**

### Kape Inato Cafe Management System
A full-stack PHP/MySQLi web system built for a real local café in Tagbilaran, Bohol. Combines a customer-facing QR-based ordering portal with a secure admin dashboard for inventory, sales analytics, and order management — including bcrypt security, PHPMailer email confirmations, and GCash/Maya QR payment with proof-of-payment upload.

**[📁 Source Code](https://drive.google.com/drive/u/1/folders/1C8-FlzOtFIf42h5cdC_Se1Aa4Qq9IAmN)**
